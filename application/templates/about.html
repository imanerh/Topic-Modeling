About Document Topic Classifier

This application uses advanced natural language processing techniques to automatically classify text documents into topics. The underlying system combines the power of:

Key Components

  • RoBERTa Embeddings: A state-of-the-art transformer model that converts text into high-dimensional semantic representations
  • UMAP Dimensionality Reduction: Reduces the high-dimensional embeddings to a manageable space while preserving important semantic relationships
  • K-means Clustering: Groups documents with similar meanings into distinct topics

How it Works

  1. When you input a document, the system generates a RoBERTa embedding that captures its semantic meaning
  2. This embedding is then projected into the same space as our pre-trained topic clusters using UMAP
  3. The system calculates the proximity to each topic cluster and assigns confidence scores
  4. The document is classified into the most likely topic, with confidence scores shown for all categories

Topics

The system currently recognizes these main topics based on the BBC News Dataset:

Business

Content about markets, companies, economy, finance, and business trends.

Entertainment

Content about movies, music, celebrities, shows, and media.

Politics

Content about government, elections, policy, and political events.

Sport

Content about games, teams, players, and sporting events.

Technology

Content about tech innovations, software, digital trends, and devices.
